# Quantum Computing Mini Project

## Overview
This project demonstrates fundamental quantum computing concepts and algorithms using Qiskit.

## Concepts Covered
- Superposition
- Entanglement (Bell State)
- Quantum Gates (H, CNOT, CZ)
- Measurement

## Algorithms Implemented
- Deutsch–Jozsa Algorithm
- Grover’s Search Algorithm

## Results
- Verified superposition with equal probability distribution
- Created entangled Bell states
- Identified balanced function using Deutsch algorithm
- Achieved search optimization using Grover’s algorithm

## Tech Stack
- Python
- Qiskit
- Matplotlib
- Jupyter Notebook

pip install -r requirements.txt
